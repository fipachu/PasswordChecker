Description

APIs, or Application Programming Interfaces, are powerful tools that allow different software applications to communicate with each other. By utilizing APIs, you can significantly expand the functionality of your programs by leveraging the capabilities of other applications or services. In this stage, you will begin interacting with APIs, specifically, the Have I Been Pwned API. This API will allow you to check whether a hashed password has appeared in a data breach.

For more information, please refer to haveibeenpwned.com.

By the end of this stage, your program will be able to interact with an API, which is a significant step towards the project's final goal.

Objectives

In this stage, you should modify your program to accomplish the following:

  1. Prompt the user to enter their password;
  2. Hash the user's password using the SHA1 hashing algorithm;
  3. Send a request to the Have I Been Pwned API using the first five characters of the hashed password;
  4. Print the sent URL indicating that the API request has been sent and the program is awaiting a response.
Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1:

Enter your password: > mypassword123
Your hashed password is: 1cf4c502ddd89b918c4bfefea76dadd590693b48
Checking...
A request was sent to "https://api.pwnedpasswords.com/range/1cf4c" endpoint, awaiting response...
