Your program should be able to interact with the user, validate their password, and implement a simple security measure by the end of this stage. You're one step closer to having a secure password checker!

Objectives

Your program should accomplish the following at this stage:

  1. Request that the user enter their password;
  2. Examine the password's length. Display a warning message that the password is too short if it is less than eight characters long. Confirm that the password has been received if it's eight characters or longer.
  3. If the password is valid, print it to the user; if not, ask the user to try again.
Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1:

Enter your password: > mypass
Your password is too short. Please enter a password of at least 8 characters.
Enter your password: > mypassword123
You entered: mypassword123

Example 2:

Enter your password: > qwerty
