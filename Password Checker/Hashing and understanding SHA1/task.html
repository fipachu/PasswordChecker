Description

The concept of hashing is one of the most fundamental in information security. Hashing is a method of encoding data that makes decoding nearly impossible without the original input. This property makes it an invaluable tool for password storage and validation. You'll learn about the SHA1 hashing algorithm and how to use it in Python at this point.

By the end of this stage, your program will securely check if a password has been compromised.

Objectives

For this stage, your program should accomplish the following tasks:

  1. Prompt the user to enter their password;
  2. Hash the user's password using the SHA1 hashing algorithm;
  3. Print the hashed password.
Example

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1:

Enter your password: > mypassword123
Your hashed password is: 1cf4c502ddd89b918c4bfefea76dadd590693b48
