Description

One of the most fundamental concepts in programming is the interaction between the user and the computer. This is usually accomplished through user input and output to the console, which you'll implement in this stage. You don't need to check the password's strength or whether it's been exposed in a data breach; you'll simply be outputting the exact password the user inputted.

At the end of this stage, your program won't do much regarding password security, but you'll have a solid foundation to build upon in the later stages of the project. Remember, every great project starts with small steps!

Objectives

Your program should accomplish the following tasks:

  1. Display a message asking the user to input their password;
  2. Take the user's password as input;
  3. Display a message confirming that the password has been received, along with the input password.
Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1:

Enter your password: > mypassword123
You entered: mypassword123

Example 2:

Enter your password: > qwerty1234
You entered: qwerty1234

Example 3:

Enter your password: > securepassword
You entered: securepassword
